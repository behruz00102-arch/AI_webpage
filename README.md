# Webpage, AI chat
🤖 AI Chat Single-Page Website

This project is a simple and efficient single-page web application that allows users to interact with an AI in a clean and user-friendly interface. It is designed for smooth navigation and quick access to conversations without unnecessary complexity.

✨ Features
💬 Chat with AI
Engage in real-time conversations with an AI model directly within the app.
📝 Rename Conversations
Easily organize your chats by renaming conversations to keep track of different topics.
🔍 Search Conversations
Quickly find past chats using the built-in search functionality.
➕ Create New Conversations
Start fresh discussions anytime with the option to add new conversations instantly.
⚙️ Usage Limitations

This project uses a free-tier API key, which comes with certain limitations:

📊 Daily Limit: The server supports up to 20 requests per day due to API constraints.
⏱️ Rate Limiting: Users can send one message every 3 seconds to prevent server overload and ensure stable performance, especially during periods of higher demand.

These measures help maintain reliability and fair usage for all users.

Instructions to install and run this website.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
